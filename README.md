# 🚙 SUV 구매 예측 프로젝트

이 프로젝트는 SUV 구매 여부를 예측하기 위한 머신러닝 분석 프로젝트입니다.  
R을 사용하여 EDA, 전처리, 시각화, 그리고 의사결정나무 모델링을 진행했습니다.

## 📁 파일 구성
- `SUV_analysis.R`: 전체 분석 코드 (EDA, 모델링 포함)

## 🔍 사용 기술
- R
- ggplot2
- rpart (의사결정나무)

## 📈 목표
SUV 구매에 영향을 주는 요인을 분석하고, 구매 여부를 예측하는 모델을 만드는 것!

## 프로젝트 목록
1. 데이터 전처리
2. 데이터 시각화
3. 의사결정나무
   1) 데이터 준비 (훈련용 데이터 + 테스트용 데이터 )
   2) 의사결정나무 모델 훈련
   3) 모델 시각화
   4) 정확도 평가
   5) 혼동 행렬 확인
   6) 변수 중요도 확인
